# samples ─ あなたの過去記事を入れるフォルダ

ここに**あなたが実際に書いた文章**を入れると、AI がそこから文体（語尾のクセ・リズム・話の運び方）を学習して、あなたの声に近い記事を書くようになります。

## 入れ方

1. 過去に書いた note 記事（またはブログ記事）の本文をコピーする
2. このフォルダに `note-1.md`（2 本目は `note-2.md`…）という名前のテキストファイルとして保存する
3. 3 本くらいあると効果がはっきり出ます。1 本でも効果はあります

ファイル形式は `.md` か `.txt`。1 ファイルにつき 1 記事。冒頭にタイトルがあると理想的ですが、本文だけでも構いません。

## 大事なルール: AI が書いた文章は入れない

ここに入れてよいのは**自分の手で書いた文章だけ**です。
AI に書いてもらった文章を入れると、AI が「AI っぽい文体」を学習してしまい、このスキルの一番の目的（AI っぽさをなくす）と逆になります。

## 選ぶコツ

- 「うまく書けた記事」より「**自分っぽい記事**」を選ぶ
- 記事のジャンルは、これから書きたいものに近いと効果的
- 空のままでも大丈夫。その場合はスキル内蔵の文体見本で書きます
