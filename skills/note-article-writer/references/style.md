# 文体スタイルリファレンス

対話調の文体で、読者と同じ目線で語りかける書き方。

> **優先順位**: `samples/` に書き手の実文がある場合は、そこから学習した文体が本ファイルより優先（本ファイルはサンプルが空のときの既定見本）。
> **テーマ別トーン**: このファイルの文体（カジュアル口語）は体験談・日常系向け。専門概念・論説系テーマでは `references/explain-hard-concepts.md` の落ち着いた論説調を優先する。

## 文体の Before / After

### 解説調 → 対話調

**Before（普通の解説調）**:
```
早起きのメリットは多くあります。まず、朝の時間は集中力が高いです。
また、誰にも邪魔されない時間を確保できます。
```

**After（対話調）**:
```
早起き、正直最初は地獄でした。

目覚ましが鳴っても「あと5分...」を3回繰り返す日々。
でも、ある日気づいたんです。

朝の1時間って、夜の3時間分くらいの価値があるんじゃないかって。

誰からも LINE 来ないし、SNS も静か。
この「誰にも邪魔されない感」が、僕にとっては革命的でした。
```

### 押し付け → 提案

**Before**:
```
毎日30分の運動をすべきです。健康のために必ず取り入れてください。
```

**After**:
```
毎日30分の運動、僕は続けられませんでした。

でも「週3回、15分だけ」にしたら、なぜか続いてる。
完璧を目指さない方が、結果的にうまくいくのかもしれません。
```

## 語尾バリエーション

### 使う語尾

| パターン | 例 |
|---------|-----|
| 〜なんですよね | これが意外と難しいんですよね |
| 〜だと思うんです | ここがポイントだと思うんです |
| 〜してみたんです | 試しにやってみたんです |
| 〜かもしれません | 合わない人もいるかもしれません |
| 〜な気がします | 本質はここにある気がします |

### 避ける語尾

| パターン | 代替 |
|---------|------|
| 〜すべきです | 〜した方がいいかも |
| 〜しなければなりません | 〜しないと困る |
| 〜でございます | 〜です / 〜なんです |
| 〜させていただきます | 〜します |

## 多義語・専門語の扱い

多義語が出たら即定義する：**「ここでいう◯◯は△△ではない。□□だ」**（否定で誤解を消し、肯定で定義を打つ）。

## 自己ツッコミの入れ方

読者の心の声を先回りして拾う：

```markdown
「いや、それって当たり前じゃん」と思いました？

僕も最初はそう思ってたんです。でも、実際にやってみると...
```

```markdown
ここまで読んで「結局、何から始めればいいの？」と思った方へ。

安心してください。次のセクションで具体的な手順を説明します。
```

## 感情表現

### 使える表現

```
正直、めちゃくちゃ焦りました
これ、地味にすごいんです
ぶっちゃけ、最初は半信半疑でした
控えめに言って、人生変わりました
```

### 避ける表現

```
非常に感動しました → 素直に感動しました
極めて重要です → すごく大事なんです
誠に恐縮ですが → すみません
```

## 段落の長さ

- 1段落は 2〜4 文を目安
- 長い段落は途中で改行を入れる
- 短い文（1文だけの段落）を効果的に使う

**効果的な短文段落の例**:
```markdown
3ヶ月、続けてみました。

人生が少しだけ変わりました。

「少しだけ」というのがポイントです。
劇的な変化じゃない。でも、確実に何かが違う。
```

## 締め方のパターン

### パターン 1: 次のアクション提示

```markdown
まずは明日の朝、5分だけ早く起きてみてください。
コーヒーを淹れる。それだけでいい。

小さく始めることが、結局は一番の近道だと思います。
```

### パターン 2: 問いかけで余韻

```markdown
完璧を目指さなくていい。
続けることだけを、目標にしてみる。

あなたは、何から始めますか？
```

### パターン 3: 共感で締める

```markdown
僕もまだまだ試行錯誤の途中です。
一緒に、少しずつ前に進んでいきましょう。
```

### 避ける締め方

```
いかがでしたか？ ← 絶対禁止
ぜひ参考にしてみてくださいね！ ← 薄い
最後までお読みいただきありがとうございました ← 形式的
```
