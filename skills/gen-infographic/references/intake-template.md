# 初回ヒアリング

図解する内容がすでに渡されている場合、次のメッセージを使う。すでに指定済みの項目は省略する。

```text
図解を生成します。以下は任意設定です。
指定したい項目だけ回答してください。
未回答の項目には記載した既定値を適用します。
「全部おまかせ」でも進められます。

1. スタイル
   既定値: 手描きマーカー（大人のグラフィックノート風）
   選べるスタイル: 手描きマーカー / ミニマル・フラット / 水彩・やわらかい / 設計図・ブループリント風
   例: 「ミニマル・フラットで」「水彩で」

2. 配色
   既定値: 選んだスタイルの標準配色に、内容に合う少数の補助色
   例: 「ブランドカラー #E86A33 を強調に使う」

3. サイズ・向き・用途
   既定値: 縦長 1024x1536、1枚で完結
   例: 「正方形、Instagram投稿向け」

4. 表現の調整
   既定値: 余白と情報量のバランスを取り、イラストは理解に必要な範囲だけ使う
   例: 「図を多め」「余白を広め」「比較を強調」

5. 参照画像の役割
   既定値: 参照画像なし
   添付する場合は、スタイル参照、イラスト参照、構成参照、内容資料のいずれかを指定できます。
   例: 「画像1はスタイル参照。線と余白だけ参考にする」

6. 必ず正確に表示する文字列
   既定値: 資料から重要語を抽出し、自然な日本語で短くまとめる
   正確性が必要な語句は引用符で囲んでください。
   例: タイトル: "生成AIを仕事で使う3つの視点"

未回答の項目には既定値を適用します。
```

図解する内容が見つからない場合は、次だけを質問する。

```text
図解したい内容を送ってください。
文章、メモ、画像、PDF、スクリーンショット、スライドのいずれでも構いません。
```
