# カードサイズ・色数・入稿仕様リファレンス

印刷物カンプの defaults とデザイン指示書に組み込むリファレンス。

## サイズ早見表（塗り足し各 3mm / 350dpi）

キャンバスは**塗り足し込みサイズ**で作る（`guides()` / `trim()` が塗り足し前提で動くため）。
画素数は「仕上がり + 塗り足し各 3mm（＝各辺 +6mm）」を 350dpi 換算した値。

| 形状 | 仕上がり | データサイズ（塗り足し込み） | 画素数 @350dpi（塗り足し込み） | 比率 |
|------|----------|---------------------------|------------------------------|------|
| しおり型 | W60×H180mm | W66×H186mm | 909×2563px | ≒1:3 |
| 名刺 | W91×H55mm | W97×H61mm | 1337×841px | ≒5:3 |
| ポストカード | W148×H100mm | W154×H106mm | 2122×1461px | ≒3:2 |
| DL（細長） | W99×H210mm | W105×H216mm | 1447×2976px | ≒3:6.4 |
| A6 フライヤー | W105×H148mm | W111×H154mm | 1530×2122px | ≒A 判 |
| 正方形カード | W91×H91mm | W97×H97mm | 1337×1337px | 1:1 |
| 二つ折り名刺（展開） | W91×H110mm | W97×H116mm | 1337×1598px | 展開後を1枚で作り中央で折る |

- **画素数 = mmpx(仕上がり mm + 6)**。`card_compose.mmpx(mm, dpi=350)` で算出（任意サイズもこれで出す）。
- **安全マージン**: 仕上がりから内側 5mm に主要要素（文字・ロゴ・QR）を収める。塗り足し込みキャンバスでは端から 3mm（塗り足し）+ 5mm = **端から 8mm** が安全ライン。`guides()` はこれをシアン線で描く。
- **仕上がり線**: 端から 3mm 内側。`guides()` はマゼンタ線で描く。`trim()` はここで切って仕上がりプレビューを返す。
- 解像度は印刷標準 **350dpi**（写真主役なら最低 300dpi）。
- 二つ折りは折り目（センター）に主要要素を跨がせない。展開データの中央に 4〜5mm の折りマージンを空ける。

## アスペクト比 → 背景生成の指定

背景画像生成時のアスペクト比は仕上がり比率に寄せる:

| 形状 | 背景生成アスペクト比 |
|------|--------------------|
| しおり型 / DL | 9:16（縦長） |
| 名刺 | 16:9 寄り（横）/ 3:2 |
| ポストカード | 3:2 |
| 正方形カード | 1:1 |
| A6 フライヤー | 3:4（縦） |

カバーフィット（`card_compose.cover`）で塗り足し込みサイズへトリミングするので、生成比率は近似で可。

## 色数（4C / 1C / 2C）の読み方と設計

色数はコストと表現を左右する最重要パラメータ。Phase 1 で必ず確定し、指示書に**表記**を明記する。

### 表記

`表面/裏面` の順でインク数を書く。

- **4C/4C** ─ 両面フルカラー（プロセス CMYK）。既定。写真主役に。
- **4C/1C** ─ 表フルカラー・裏スミ 1 色。コスト圧縮の定番（裏は情報のみ）。
- **1C/1C** ─ 両面モノクロ（スミ or 特色 1 色）。上質・端正。ファインペーパーと好相性。
- **2C**（例: `2C（特色: DIC-xxx + K）`）─ 特色 2 色刷り。ブランドカラーを正確に出す・上品な差し色。

### 色数別の入稿ルール

- **共通**: 黒文字は **K 単色**（リッチブラック＝CMYK 全盛りは文字に使わない。版ずれと潰れの原因）。写真の黒はリッチブラック可。
- **1C**: 階調は K の網 % で作る（100 / 70 / 40 / 10〜20）。写真は `to_mono(autocontrast=True)` でコントラストを立てる（そのままだと眠い）。**QR は K100 単色**。
- **2C**: インクの役割を分ける ─ 濃色インク＝文字・写真の duotone / 特色＝アクセントと地。中間調は各インクの網 % で作る。背景写真は**グレースケールで生成し `duotone()` で色付け**する（プロンプトで 2 色指定するとインク色がブレる）。特色は **DIC / PANTONE 番号**で指示書に明記。**QR は濃い方のインク単色**。
- **4C**: 自由。ただし黒文字は K 単色を守る。特色を併用する場合は 4C+特色＝5C 扱いでコスト増を申し送る。

### CMYK で沈む色（指示書に申し送る）

蛍光ピンク・鮮やかなエメラルド・鮮やかな青紫・ネオン系は CMYK で再現できず沈む。RGB カンプで映えても実物は濁るため、該当色を使う場合は指示書の申し送り欄に明記し、色校正を促す。

## 入稿データ チェックリスト（指示書 .md に転記する土台）

- [ ] データサイズ = 仕上がり + 塗り足し各 3mm／トンボ付き（塗り足し込みキャンバスで制作済み）
- [ ] 主要要素は仕上がりから 5mm 内側（塗り足し込みでは端から 8mm）に収まっている
- [ ] **色数表記を明記**（4C/4C・4C/1C・1C/1C・2C など）／特色は DIC・PANTONE 番号を記載
- [ ] 画像は 350dpi／カラーモード CMYK（1C・2C はグレースケール or 特色版に分版）
- [ ] **黒文字は K 単色**（リッチブラック不使用）／写真の黒はリッチブラック可
- [ ] フォントはアウトライン化（入稿時）
- [ ] ロゴは支給ベクター（VI）データに差し替え済み（カンプの生成ロゴは入稿不可）
- [ ] QR はクワイエットゾーン（周囲余白）を確保し **15mm 角以上**（≒ `mmpx(15)` = 207px@350dpi）で配置、実機スキャン確認済み／1C・2C では濃色インク単色
- [ ] アクセントカラー（ブランド指定色）・特色は実物と色校正／CMYK で沈む色を申し送り済み
- [ ] 折り加工がある場合、折り目に主要要素が跨っていない

> [!note] カンプ → 入稿の境界
> 本スキルが出すのは **カンプ（PNG）+ 指示書**まで。トンボ付き入稿データ化・分版・アウトライン化・色校正は Illustrator/Figma での別工程（スキル範囲外）。塗り足し込みキャンバスで作るのは、後工程が座標を再現しやすくするため。

## 用紙の定番（指示書で提案する候補）

| 用紙 | 質感・発色 | コスト | 相性の良い色数・用途 |
|------|-----------|--------|--------------------|
| マットコート 220kg | 発色と質感のバランス・汎用 | 安 | 4C 写真主役。名刺・ポストカード全般 |
| コート／アートポスト | 発色最良・光沢 | 安〜中 | 4C 写真・ビジュアル重視 |
| ファインペーパー（ヴァンヌーボ等） | ナチュラルな手触り・上質感 | 中 | 1C・2C・世界観重視。しおり・ショップカード |
| 色上質・エンボス系 | 温もり最大・写真の精細感は落ちる | 中 | 1C・2C。和・ナチュラル系 |
| クラフト紙 | 素朴・地色が濃い（**白インキなしだと白が出ない**） | 中 | 2C の差し色・ロゴ主体。地色を活かす設計に |

- しおり等「手に残す」用途は厚め（220kg 以上）+ 角丸 + マット PP で質感を強化（任意）。
- **角丸の目安**: R3〜5mm。指示書に半径を明記（`R3` 等）。
- クラフト・色上質は白が沈むため、白抜き文字・白ロゴは避けるか白インキ（特色白）を申し送る。
