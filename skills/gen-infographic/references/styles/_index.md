# 共有スタイルライブラリ（_image-styles）

画像生成系スキルが横断で参照する「スタイル指定ブロック」のライブラリ。

各ファイルは **構造非依存**。レイアウトや情報設計（流れ図・対比・循環など）には一切触れず、見た目（線・配色・質感・筆致）だけを定義する。これにより、同じ構造をスタイルだけ差し替えて生成できる。

## 収録スタイル

| ファイル | スタイル | 向く用途 |
|---|---|---|
| `handdrawn-marker.md` | 大人の手描きマーカー（既定） | SNS 解説画像、親しみやすい図解 |
| `minimal-flat.md` | ミニマル・フラット | ビジネス資料、清潔感のある図解 |
| `watercolor.md` | 水彩・やわらかい | 情緒的・あたたかい印象の解説画像 |
| `blueprint.md` | 設計図・ブループリント風 | 技術・構造説明、設計っぽい図解 |

## 使い方（参照側スキル）

骨格プロンプト（`generation-prompt.md` 等）の `{{STYLE}}` プレースホルダーに、選ばれたスタイルファイルの本文を差し込む。スタイル未指定なら `handdrawn-marker.md`（既定）を使う。

## 共有の仕組みと注意

- 正本はこのディレクトリ。各スキルの `references/styles/` はここへの相対 symlink。
- 編集はこの正本1箇所で行う。各スキルに反映される。
- `.skill` としてパッケージ配布する場合は symlink が同梱されないため、各スキル側の `scripts/sync-styles.sh` で実体コピーに展開してからパッケージ化する。
