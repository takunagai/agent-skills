# 修正用プロンプト

直前に生成した画像を入力画像として渡し、次のテンプレートを埋め込む。

```text
直前の図解画像を編集してください。

変更するのは、以下の「今回の変更」だけです。
それ以外は維持してください。

# 必ず維持するもの
- 全体のレイアウト
- 情報の階層
- 読む順番
- 矢印、囲み線、下線の位置
- 配色と色の役割
- 紙の質感
- 大人の手描きマーカーという印象
- 修正対象ではない日本語テキスト
- 修正対象ではない数値と固有名詞
- 余白
- 画像サイズと向き

# 文字
修正する文字列は、引用符の内側を一字一句そのまま表示してください。
余計な文字を追加しないでください。

# 今回の変更
---
[今回の変更]
---

# 出力
編集後の画像だけを生成してください。
説明文は出力しないでください。
```
