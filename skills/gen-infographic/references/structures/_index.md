# 構造ブロック（structures）

図解のレイアウトと情報設計を定義するブロック群。各ファイルは **スタイル非依存**で、線の太さや配色などの見た目には触れない。内容に応じて適切な構造を1つ選び、骨格プロンプトの `{{STRUCTURE}}` に差し込む。

## 構造の選び方

| 内容の性質 | 使う構造 | ファイル |
|---|---|---|
| 手順・時系列・ステップ | 流れ図 | `flow.md` |
| 2つ（以上）の比較・対比 | 左右/上下の対比 | `contrast.md` |
| 中心概念と要素・繰り返す仕組み | 循環・放射 | `cycle.md` |
| 多数の項目の一覧・分類 | グループ分け一覧 | `grouped-list.md` |

内容が複数の性質を持つ場合は、主たる構造を1つ選び、必要なら補助的に組み合わせる。構造が内容に合わないと感じたら、無理に当てはめず内容に適した構造を自律的に設計する。
