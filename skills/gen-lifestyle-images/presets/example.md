# Brand Guidelines

## ターゲット
丁寧な暮らしを志向し、物の背景や品質にこだわりを持つ層。（自分のブランドのターゲット像に置き換える）

## ブランドパーソナリティ
- **3つの言葉**: 上質・ナチュラル・洗練
- **声のトーン**: 押しつけがましくなく、静かに自信を持って語りかける
- **感情目標**: 本物を選んだ満足感

## アンチパターン
ギラギラした派手さ / カジュアルすぎるポップさ / 冷たいテック感 / 重厚すぎる企業サイト感

# Photography Style

## 光源・色温度
- 窓からの柔らかい自然光（側光〜逆光気味）
- 直射光・フラッシュ・人工光のギラつきは避ける
- やや暖色（5500-5800K）。白〜ベージュ基調

## カメラ
- 低めの目線（10-35°、シーンにより変動）
- 被写界深度: 浅め（f/1.8-3.5、シーンにより変動）
- 商品が主役になる構図

## スケール感
周囲の家具・小物と不自然な大小差が出ないよう実物大に近い比率を維持

# Styling Guide（商品タイプ別）

着用シーン用のスタイリング指示（服装・ソックス・小物）を商品タイプ別に記述する。

- **定番モデル**: ライトグレー or 白のパンツ + 白 or 生成りのソックス
- **限定/アップリケ系**: ニュートラルカラーのパンツ + 白 or ライトグレーのソックス。デザインが映える服装

# Scene Library

シーン定義のライブラリ。各シーンに `id`・名前・時間帯・構図・小物・光源・カメラ設定・相性の良い商品タイプを記述する。3 タイプ（着用 / ディテール / 複数アイテム）に分類する。以下は汎用シーンの例。

## Type A: 着用ライフスタイル

### A1: 朝のキッチン — コーヒーを淹れるひととき
- **時間帯**: 平日の朝 7:00-8:00
- **構図**: 膝下フレーミング。キッチンの床に立つ足元。カウンター脚部がフレーム左に見切れる
- **小物**: 白タイル壁、木製カウンター脚、リネンキッチンクロス、床に落ちる窓の光
- **光源**: 東向き窓からの朝の斜光、5800K
- **カメラ**: 30°、やや横プロフィール。f/2.8
- **相性**: 暖色系・カラフルな商品

### A2: 午後のリビング — 読書のひと休み
- **時間帯**: 休日午後 14:00-15:00
- **構図**: 膝下フレーミング。ソファに座り足を前に投げ出す
- **小物**: 無垢オーク材フロア、リネンソファ脚部、伏せた文庫本、白マグカップ
- **光源**: 南向き大窓からの拡散光、5500K
- **カメラ**: 35°、正面やや右。f/3.2
- **相性**: 落ち着いた色味

### A3: 夕方の窓辺 — 日が傾く静かな時間
- **時間帯**: 夕方 16:00-17:00
- **構図**: 膝下フレーミング。大窓の前に立つ足元。逆光気味
- **小物**: 薄手リネンカーテン、窓台に陶器の一輪挿し、淡色フロア
- **光源**: 西日の逆光、リムライト効果、5500K
- **カメラ**: 10-15°低め。f/2.8
- **相性**: ライトカラー・繊細なデザイン

## Type B: ディテールクローズアップ

### B1: 素材質感マクロ
- **構図**: 商品表面の極接写。素材のテクスチャ、ステッチ、トリム
- **光源**: サイドライト、質感の 3D 感を強調、5800K
- **カメラ**: マクロ寄り。f/1.8-2.0 極浅 DOF
- **相性**: 素材感が魅力の商品

### B2: 甲デザイン俯瞰
- **構図**: 真上からの俯瞰ショット。ペアを自然に配置、デザインが明確に見える
- **小物**: ライトオーク材フロア
- **光源**: 斜め窓光、5600K
- **カメラ**: 真上〜45°。f/2.8
- **相性**: アップリケ・デザイン系

## Type C: 複数アイテム

### C1: カラーコレクション
- **構図**: 同シリーズ 2-3 色を斜めの対角線上に配置。各ペアはハの字
- **小物**: ライトオーク材フロア、リネンクロス、小さなグリーン
- **光源**: 窓からのサイドライト、5600K
- **カメラ**: 45°上から。f/3.2
- **着用**: なし
- **相性**: 同モデル色違い

# Avoid（避けるべき要素）

- 不自然なテキスト・ロゴ・ウォーターマークの生成
- 過度に彩度の高い加工感のある色味
- 商品の形状の歪み、左右非対称
- 生活感がありすぎる散らかった背景
- 商品の色味が元写真と大きく異なること
- ブランドラベルのテキストが不正確な生成（リファレンス画像を必ず参照）
