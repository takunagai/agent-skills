# デザイン原則リファレンス（なぜそう組むか）

印刷物カンプを設計する際、AI が「なぜその余白・書体・配色・階層で組むか」を判断するための正本。コンセプト提案（Phase 1）と PIL 合成（Phase 4-5）の両方で参照する。数値目安は `references/card-specs.md`（サイズ・色数の正本）に整合させてある。

## 1. 余白システム

余白は「余り」ではなく設計要素。詰め込みは素人臭さの最大要因なので、まず余白を確保してから要素を置く。

- **外周マージン（デザイン余白）**: 仕上がり短辺の **8% 以上**を目安に、版面（要素を置く内枠）を切る。名刺（短辺 55mm）で約 4.4mm、ポストカード（短辺 100mm）で 8mm、しおり型（短辺 60mm）で約 4.8mm。写真を裁ち落とす面はこの限りではないが、文字・ロゴ・QR は必ず版面の内側に置く。
- **安全マージンとの区別**: 安全マージン 5mm（塗り足し込みキャンバスでは端から 8mm）は「断裁でも切れない最低ライン」であって、デザイン余白ではない。安全ライン上に文字を置くのは「切れないが窮屈」な状態。デザイン余白 = 安全ライン + α（上記 8%）で考える。
- **要素間隔は 大:中:小 の 3 階層**をつける（近接の原則）。関連する要素は近く、グループが変わる境目は広く。目安比率は **小 : 中 : 大 ≒ 1 : 2 : 4**（例: 行内 4px / 項目間 8px / グループ間 16px、あるいは 1.5mm / 3mm / 6mm）。同じ間隔を全箇所に均等配分するとグループ構造が消える。
- **占有率の上限**: 版面に対する要素の占有面積は **50〜70% 程度**に抑え、残りを余白に回す。情報が多い裏面でも 75% を超えたら文字サイズを下げるより情報を削る・2 段組にするを先に検討する。

版面（デザイン余白の内枠）の早見表（8% 目安。塗り足し込みキャンバスの端からの距離 = デザイン余白 + 塗り足し 3mm）:

| 形状 | 仕上がり短辺 | デザイン余白（8%） | キャンバス端からの距離 |
|---|---|---|---|
| 名刺 | 55mm | 約 4.4mm（安全 5mm を採用） | 約 8mm |
| ポストカード | 100mm | 8mm | 11mm |
| しおり型 | 60mm | 約 4.8mm（安全 5mm を採用） | 8mm |
| A6 フライヤー | 105mm | 約 8.4mm | 約 11mm |

- 8% が安全マージン 5mm を下回る小型カード（名刺・しおり）は、**安全マージン 5mm 側を採用**する（切れないラインを優先）。8% が 5mm を上回る中判以上はデザイン余白を優先する。

## 2. タイポグラフィ

- **明朝 × ゴシックのペアリング**が基本。役割で分ける ─ 情緒・世界観・見出し・キャッチは**明朝**（`mincho`）、実務情報・視認性が要る TEL/URL/住所・注記は**ゴシック**（`gothic_w3` / 太字は `gothic_w6`）。1 カンプで書体ファミリは 2 種までに抑える。
- **丸ゴシック（`maru`）の使いどころ**: ポップ・食品・子供向け・親しみを出したいときに限定して使う。コーポレート・ラグジュアリー・和の端正さには合わない。
- **縦組み（`tate`）の使いどころ**: 和・料亭・呉服・しおりなど格式・情緒を出す面で有効。縦組みは明朝と相性がよく、右→左に読ませる。数字・欧文・URL は横組みが読みやすいので、和の縦組み面でも連絡先ブロックだけ横組みに分けるのが定石。
- **ジャンプ率（見出し : 本文の大きさ比）**: メリハリ型は **2.5〜4 : 1**（訴求を強くしたい表面・広告的）、上品型は **1.5〜2 : 1**（世界観・落ち着き重視）。例: 本文 36px なら、メリハリ型で見出し 90〜144px、上品型で 54〜72px。
- **字間 tracking（`tx` / `tate` / `fit_font` の px 引数）**: 日本語見出しは `tracking = フォントサイズ × 0.02〜0.08`（px）を目安（72px 見出しなら 1.4〜5.8px）。本文はベタ〜+2%、欧文は基本ベタ。**字間を空けると上品・広告的、詰めると力強い**。
- **行間 line_height（`tx` の引数）**: 本文 **1.7〜2.0**、見出し **1.2〜1.4**。行が詰まると窮屈、空きすぎると散漫。
- **1 行の適正字数**: 横組み和文本文は **15〜35 字**が読みやすい範囲。これを超えたら改行位置を設計するか 2 段組を検討する。カード類は幅が狭いので、長文はそもそも載せず要点に絞る。

ジャンル別の書体ペアリング早見表（見出し＝情緒／本文＝視認性の役割分担）:

| ジャンル | 見出し・キャッチ | 本文・情報 | 備考 |
|---|---|---|---|
| 和・上品 | 明朝（`mincho`） | 明朝 or 細ゴシック（`gothic_w3`） | 字間を広め（× 0.05〜0.08）にして格を出す |
| ナチュラル | 明朝 | 細ゴシック | ジャンプ率は上品型（1.5〜2 : 1）に寄せる |
| コーポレート | 太ゴシック（`gothic_w6`） | 細ゴシック | 明朝見出しでも可。堅実さ重視で字間はベタ〜+2% |
| ポップ・食品 | 丸ゴシック（`maru`） | 丸ゴシック or 太ゴシック | ジャンプ率を高め（2.5〜4 : 1）にして元気を出す |
| ラグジュアリー | 明朝（細ウェイト） | 細ゴシック（欧文中心なら serif） | 字間を広く取り、余白を多めに |

## 3. 配色

- **70-25-5 バランス**: 基調色 70% / 従色 25% / アクセント 5% で面積配分する。基調は地・余白、従色は罫線・パネル・見出し地、アクセントは CTA・強調の一点。
- **アクセントは 2 箇所まで**に絞ると効く（例: 区切り線 + CTA バッジ）。3 箇所以上に散らすと「どれも目立たない」状態になり効果が死ぬ。
- **70-25-5 のカード裏面への当てはめ例**: 生成り地・余白が 70%（基調）、社名や罫線・パネルの地が 25%（従色）、CTA バッジと区切り線だけが 5%（アクセント）。この面積配分を崩してアクセント色の面積が増えると、視線の落ち所がぼやける。
- **スミ文字は真っ黒（#000000）にしない**。`#282624` 系（RGB 40,38,36 のダークウォームグレー）で刷ると、紙の上で硬くならず柔らかく馴染む。`card_compose` のレシピでは `INK = (40, 38, 36, 255)` が既定。
- **ジャンル別パレット例**（基調 = 70% の地、アクセント = 5% の一点。CMYK は近似値）:

| ジャンル | 用途 | 基調色 | アクセント |
|---|---|---|---|
| 和 | 和菓子・呉服・料亭・ショップカード | 生成り地 RGB(245,241,233) / CMYK(3,4,10,0) | 藍 RGB(35,54,95)/CMYK(90,75,30,15)・臙脂 RGB(158,42,52)/CMYK(30,90,75,20) |
| ナチュラル | 雑貨・カフェ・オーガニック | オフホワイト RGB(247,243,236) / CMYK(4,5,10,0) | テラコッタ RGB(192,105,75)/CMYK(22,68,72,5)・オリーブ RGB(122,120,74)/CMYK(52,42,80,15) |
| コーポレート | 士業・IT・BtoB 名刺 | 白 RGB(255,255,255) / CMYK(0,0,0,0) | ネイビー RGB(28,44,79)/CMYK(95,85,35,25)・グレー RGB(120,124,130)/CMYK(30,22,18,10) |
| ラグジュアリー | サロン・宝飾・高級店 | 黒/濃紺 RGB(20,24,38) / CMYK(88,82,55,60) | ゴールド RGB(176,141,87)/CMYK(30,42,72,10)・オフ白 RGB(238,232,220)/CMYK(7,8,14,0) |
| ポップ | 食品・イベント・子供向け | 白 RGB(255,255,255) / CMYK(0,0,0,0) | ビビッドイエロー RGB(255,205,0)/CMYK(0,18,95,0)・マゼンタ RGB(230,30,110)/CMYK(5,90,25,0) |

- **本文と地の明度コントラスト**: 可読性は色相差より明度差で決まる。本文文字は地との明度差を大きく取る（濃色地 × オフ白文字、生成り地 × ダークウォームグレー文字など）。装飾・従色は明度差が小さくても許容できるが、TEL/URL/注記のような読ませる要素は明度差を確保する。半透明パネル（`panel`）を敷いて文字の地を安定させるのも有効。
- **CMYK で沈む色を申し送る**: 蛍光ピンク・鮮やかなエメラルド・鮮やかな青紫・ネオン系は CMYK で再現できず濁る。RGB カンプで映えても実物は沈むため、該当色を使う場合は指示書の申し送り欄に明記し色校正を促す（詳細は `card-specs.md` の「CMYK で沈む色」節）。ポップのビビッド 2 色も彩度が高いほど沈むので注意。

## 4. 情報階層と視線誘導

- **最重要要素は 1 つに絞る**（主役の明確化）。主役はサイズ・余白・色のいずれか（できれば複数）で他を明確に上回らせる。主役候補が 2 つ以上あるカンプは、ユーザーに優先順位を確認してから組む。
- **視覚的中心はやや上**: センター軸で主役を縦位置に置くとき、幾何学的中心（高さの 50%）ちょうどより **2〜5% 上**に置くと安定して見える（下すぎると沈んで見える）。ロゴやキャッチを中央配置する `pc` / `tc` の y 座標はこの補正を意識する。
- **視線導線の使い分け**:
  - **Z 型** ─ 横長・要素少（名刺表など）。左上→右上→左下→右下に主要素を配置。
  - **F 型** ─ 情報量が多い縦長（フライヤー裏・DL カード）。左揃えで見出し→本文を段階的に読ませる。
  - **センター軸** ─ ブランド訴求・しおり・ショップカード。中央 1 本の軸に主役を通し、静けさと格を出す。
- **裏面（情報面）は揃え軸を 1 本決めて全要素を従わせる**。中央軸なら全要素を中央（`tc`）、左軸なら全要素を左（`tx` で x 座標を統一）に揃える。中央と左が混在すると素人臭さが一気に出る。揃え軸のズレは px 単位で一致させる（同じ左マージンなら全行が同一 x で始まる）。

カード種別と推奨導線の対応（`layout-patterns.md` の型選定と揃える）:

| 形状 | 表面（訴求面） | 裏面（情報面） |
|---|---|---|
| 名刺 | Z 型（横長・要素少） | 左軸 F 型（社名→役職→連絡先→QR） |
| しおり型・ショップカード | センター軸 | センター軸 |
| ポストカード | Z 型 or 写真全面 | 左軸 F 型 |
| DL・A6 フライヤー | センター軸 or F 型 | 左軸 F 型（情報量が多い） |

## 5. 色数別の設計指針

`card-specs.md` の色数節と矛盾させないこと。括弧内は `card_compose` の該当ヘルパ。

- **1C（モノクロ 1 色）**: 階調は K 濃度 **100 / 70 / 40 / 10〜20** の網 % で設計する。写真は `to_mono(autocontrast=True)` でコントラストを立てる（そのままだと眠い）。QR は **K100 単色**。ガイド確認は `guides()` / 仕上がりプレビューは `trim()`。
- **2C（特色 2 色刷り）**: インクの役割を分ける ─ **濃色インク = 文字・写真の `duotone()`** / **特色 = アクセントと地**。中間調は各インクの網 % で作る。背景写真は**グレースケールで生成し `duotone()` で色付け**する（プロンプトで 2 色指定するとインク色がブレる）。特色は **DIC / PANTONE 番号**を指示書に明記。QR は**濃い方のインク単色**。
- **4C（プロセスフルカラー）**: 配色は自由。ただし**黒文字は K 単色**（リッチブラック＝CMYK 全盛りは文字に使わない。版ずれと潰れの原因）。写真の黒はリッチブラック可。

網点の実用範囲（1C・2C の階調設計で重要）: 網 **5% 以下は印刷で飛ぶ**（紙の白に埋もれる）、**95% 以上はベタに潰れる**（ドットゲイン）。淡い階調は最低 10% 以上、濃い階調は最高 90% 程度で止め、100% はベタ塗り専用と考える。1C の 4 段階（100 / 70 / 40 / 10〜20）はこの実用範囲に収まるよう設計してある。

色数別の要点早見表:

| 色数 | 写真の扱い | 中間調 | QR | 主なヘルパ |
|---|---|---|---|---|
| 1C | `to_mono(autocontrast=True)` | K 網 % 100/70/40/10〜20 | K100 単色 | `to_mono` / `guides` / `trim` |
| 2C | グレースケール生成 → `duotone()` | 各インクの網 % | 濃い方のインク単色 | `duotone` / `to_mono` |
| 4C | 自由（黒文字のみ K 単色） | CMYK 自由 | K100 単色 | `cover` / `panel` / `badge` |

## 6. 紙との関係

- **色上質・クラフト紙は白が出ない**（白インキなし前提）。地色が濃いため、**白抜き文字・白ロゴは避ける**か、必要なら特色白（不透明白）を指示書に申し送る。地色を活かす設計（濃色地に濃インクの版）に寄せる。
- **写真主役はコート系**（マットコート 220kg・アートポスト）＝発色最良。**1C・2C はファインペーパー（ヴァンヌーボ等）や色上質と好相性**で、上質・端正な質感が出る。
- **厚み・加工が「手に残る」質感を決める**。しおり等の手元に残す用途は **220kg 以上**の厚みを目安に、**角丸 R3〜5mm**（指示書に半径を明記）・**マット PP** で触感を強化する。加工は任意だが、質感訴求の案件では積極的に候補として提案する。

紙と色数・演出の相性（`card-specs.md` の用紙定番表と併読）:

- **写真主役 4C** → コート／アートポスト（発色最良・光沢）。マットにしたいならマットコート 220kg。
- **1C・2C の世界観重視** → ファインペーパー・色上質（手触りと上質感）。白抜きが必要なら白が出る明るい地の紙を選ぶ。
- **地色を活かすデザイン** → クラフト・色上質。濃インクの版で組み、白抜きは避ける（前項）。
- **加工の目安**: 角丸は R3〜5mm（小型カードは R3、ポストカード以上は R5 まで）。マット PP は指紋・擦れに強く高級感が出る。グロス PP は写真の発色をさらに立てるが安っぽく見える場合もあるので用途を選ぶ。
