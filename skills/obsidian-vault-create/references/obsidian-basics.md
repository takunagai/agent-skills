# Obsidian Basics (Official docs summary)

- An Obsidian vault is just a folder on disk that stores Markdown files; subfolders are allowed.
- Each vault stores its settings in a `.obsidian` folder inside the vault.
- Avoid nesting vaults inside other vaults.
- Core plugins are built-in; "Templates" and "Daily notes" are core plugins that can be enabled in Settings.
- Templates support variables like `{{date}}` and `{{time}}`, with optional date formats (e.g., `{{date:YYYY-MM-DD}}`).
