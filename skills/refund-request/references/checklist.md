# 送信前チェックリスト

メール送信前に以下を確認し、ユーザーに提示する。

---

## 必須チェック項目

### 📋 メール内容の確認

- [ ] **件名に Transaction ID / 注文番号が含まれている**
- [ ] **送信先アドレスが正しい**（企業の公式サポートアドレス）
- [ ] **送信元メールアドレスがアカウント登録メールと一致している**（違うアドレスから送ると対応してもらえない可能性がある）
- [ ] **自分の名前**がアカウント登録名と一致している
- [ ] **金額が正確**（課金明細と照合）
- [ ] **日付が正確**（課金日、購入日）
- [ ] **Transaction ID / 注文番号が正確**（コピペで入力）
- [ ] **具体的な希望が明記されている**（全額返金 / 部分返金 / クレジット等）
- [ ] **感情的な表現がない**（怒りや非難ではなく、事実ベース）
- [ ] **誤字脱字がない**

### 🔧 事前アクションの確認

- [ ] **自動更新を停止した**（サブスクの場合）
- [ ] **課金明細のスクリーンショットを保存した**
- [ ] **アカウントの利用状況画面をスクリーンショットした**（未使用の証明）
- [ ] **関連するメール（更新通知等）を確認した**
- [ ] **迷惑メールフォルダも確認した**（通知未受領を主張する場合）

### 📎 添付ファイルの確認

- [ ] **必要な証拠を添付した**（スクリーンショット等）
- [ ] **添付ファイルに個人情報が過剰に含まれていない**（クレジットカード番号等はマスク）
- [ ] **ファイルサイズが適切**（メールの添付上限を超えていない）

### 📝 利用規約の確認

- [ ] **メールで引用した返金ポリシーの内容が正確**（原文と照合）
- [ ] **返金申請期限内である**
- [ ] **引用元の URL をメモしている**（後から参照できるように）

---

## 推奨チェック項目

### 💡 戦略面の確認

- [ ] **最初のメールで代替案に言及していない**（全額返金の可能性を最大化）
  - ※ ユーザーが意図的に代替案を含めた場合は OK
- [ ] **返信期限の目安を設定した**（自分の中で: 3営業日 → 1週間 → フォローアップ）
- [ ] **メールの控え / 送信済みフォルダを確認できる状態**
- [ ] **クレジットカード会社の chargeback 期限を把握している**（最終手段のバックアップ）

### 🌐 言語面の確認（英語メールの場合）

- [ ] **ネイティブに自然な表現になっている**
- [ ] **ビジネスメールとして適切な丁寧さ**
- [ ] **略語や俗語を使っていない**
